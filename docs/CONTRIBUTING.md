# How to contribute to Quick Scope

This guideline contains several steps to help you contribute to [Intigriti Quick Scope](https://github.com/intigriti/iqs). Thank you for helping out. We appreciate all contributions made to this project!

## New feature requests

Want to help with introducing a new feature or improving an existing one? Please create a pull request and help us by preserving our coding style. This includes adding detailed comments to each new function, applying coding best practices, and, of course, indenting your code with tab characters.

> [!INFO]
> If you cannot code and still wish to see your idea come to life, feel free to create a [GitHub issue](https://github.com/intigriti/iqs/issues/new/choose) instead. We welcome all contributions!

## Reporting bugs

If you encounter any unexpected behavior, you may always [open a new issue](https://github.com/intigriti/iqs/issues/new/choose).

Please include as much information as possible to help us understand the root cause of it.

> [!WARNING]
> Security bugs should be reported to our security contact at [community@intigriti.com](mailto:community@intigriti.com)!
