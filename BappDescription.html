Intigriti Quick Scope (IQS) is an extension that automates bug bounty project and scope setup by fetching program data directly from the Intigriti Researcher API. It allows researchers to import in-scope and out-of-scope domains from their Intigriti programs, including private ones, without manual configuration.

Features

  • Fetches all available Intigriti programs, including private programs, using your API credentials.
  • Applies an entire programs scope configuration in a single click.
  • Organizes domains into tabs by type: Web, Others (iOS, Android, source code repositories, etc.), and Out of Scope.
  • Supports adding individual selected domains to the active scope rather than importing all at once.
  • Automatically inserts your username into required scope headers when a program mandates them.
  • Displays program-level scope requirements such as mandatory request headers and rate limits.
  • Provides a Reset Scope button to clear all scope filters and headers and start fresh with a different program.

Usage

  1. Open the IQS tab and enter your Intigriti username and API key in the configuration fields.
  2. Click Test Researcher API to verify your credentials are working correctly.
  3. Click Load Programs to retrieve your available programs from the Intigriti API.
  4. Use the filters to narrow the program list by active, followed, or private status.
  5. Select a program to view its domain details, organized across the Web, Others, and Out of Scope tabs.
  6. Click individual domains to add them selectively to your scope, or apply the full program scope at once.
  7. If the program requires scope headers, confirm that your username has been automatically inserted into the relevant header fields.
  8. To switch programs or start over, click Reset Scope to clear all current scope filters and headers.
